Верстка “Форест тревел”

Описание задачи

Сверстать все страницы сайта точно по макету. Выполнить мобильную адаптацию. Провести оптимизацию загрузки.

Решение

Сначала были свёртаны все страницы сайта. Настроены необходимые гиперссылки для навигации по сайту. Интегрированная интерактивная карта с помощью сервиса "Яндекс Карты".

Следующим шагом стала адаптация свёрстанных страниц под различные устройства. И тут я столкнулся с трудностью адаптации страницы - "Туры и направления". Размер всех карточек разный и соотвесенно размеры изображений в этих карточках тоже разные. Чтобы карточки не набегали друг на друга, а изображения не обрезались безконтрольно. Я реструктуризировал код для более элегантного взаимодействия с объектами.

После окончания работ по адаптации сайта пришло время оптимизации. Все изображения были сжаты по весу, без зрительной потери качества. Были установлены необходимые правила загрузки изображений для разных размеров экранов.

Результат
В результате получилась рабочая страница туристического агенства
